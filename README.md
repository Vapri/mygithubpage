Installation
--------------

```
//make sure you have all req dep installed
npm -v
node -v
bower -v
//Otherwise Install Nodejs and do the following 
npm install -g bower

//Next Navigate to the folder
cd D:\directoryLocation

//setup the toolbox
npm install
bower install

//start the toolbox
grunt serve

//build the site and test it (this may take a bit of time)
grunt serve:dist

```
